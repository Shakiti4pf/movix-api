# 🎬 Movix API

A professional Movie Database REST API built with Node.js, Express, PostgreSQL and JWT Authentication.

## Features

- User Authentication
- JWT Security
- Movies CRUD
- Genres
- Actors
- Directors
- Ratings
- Reviews
- Watchlist
- Favorites
- Recommendations
- Poster Upload
- Swagger Documentation

## Tech Stack

- Node.js
- Express.js
- PostgreSQL
- JWT
- Multer
- Swagger UI

## Installation

```bash
npm install
npm start
```
